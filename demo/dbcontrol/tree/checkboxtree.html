Jslet - DBTreeView(CheckBox) Demo

Correlate checkNon-correlate checkItem Checkable(Only the leaf item has check box)

Demo Content:

  1. After check a child node, all of its parent node will display bold, it is very useful to find all checked nodes when the parent nodes are collapsed.
  2. Correlate check parent node and child node when check on a tree node.
  3. Check tree node separately. You can right click on a tree node to select/unselect all.
  4. Set which tree node has check box. In this sample, only leaf tree node has check box.
